# RFID 125kHz (AU Focus)

125kHz RFID is still everywhere in Australia.

## Common AU Targets
- Apartment building / garage access
- Office / warehouse swipe cards (many still use 125kHz Prox / EM4100 / HID)
- Carpark boom gates
- Older alarm systems and immobilisers
- Gyms, storage units, rural properties

## Popular Formats in Australia
- EM4100 / EM4102 (very common)
- HID Prox (26-bit and others)
- Indala
- Some Awid, Keri, etc.

## Flipper Strengths Here
- Excellent 125kHz support
- Can read many cards that phones can't
- Can emulate and write to T5577 / EM4305 blanks (very useful in AU)

## Recommended Blanks to Carry
- T5577 (most flexible)
- EM4305

## Pro Tip
In Australia, a lot of "access control" companies still install cheap 125kHz systems in 2024-2026 because they're "reliable and cheap". This is your advantage.
