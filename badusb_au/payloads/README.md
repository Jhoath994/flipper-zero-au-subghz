# BadUSB Payloads - Australian Focus

**All payloads in this folder are for authorised testing and educational purposes only.**

Using these without explicit permission on systems you do not own is illegal in Australia and will get you charged.

## Folder Structure
- `windows/` - Windows-targeted payloads (most common in AU)
- `macos/` - macOS (less common but growing)
- `linux/` - Linux (rare in tradie environments)

## Australian Social Engineering Themes That Still Work

- "Telstra / Optus bill overdue - immediate action"

## Important Notes
- Good pretext beats fancy code 95% of the time in Australia.
- Tradie laptops and small business machines are often the softest targets.
- Many still run Windows 10/11 with minimal protection and click "Yes" on everything.
