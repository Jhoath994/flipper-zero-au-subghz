# NFC (MIFARE, NTAG, etc.) - Australian Focus

## Relevance in Australia
- Public transport cards in some states still have legacy MIFARE Classic implementations (research only).
- Building access, gyms, universities, and some corporate sites use MIFARE.
- NTAG215/216 are extremely common for "tap to unlock" and marketing tags.

## Current State (2026)
MIFARE Classic attacks are well understood. Many older Australian systems are still vulnerable to classic attacks (nested, darkside, etc.) if the keys are weak or default.

Newer systems have moved to better crypto, but legacy deployments remain.
