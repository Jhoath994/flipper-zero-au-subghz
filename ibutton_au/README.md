# iButton / Dallas 1-Wire (AU Focus)

This is one of the highest value, most under-appreciated attack vectors for Flipper users in Australia.

## Why It Matters in Australia
- Extremely common on trucks, utes, and heavy machinery (especially older ones).
- Used on depot gates, toolboxes, fuel bowsers, workshop access.
- Many older ATA / B&D / Elsema gate systems still use iButton for master access.
- Tradies and fleet operators love them because they're "indestructible".

## Common AU Use Cases
- Truck & machinery immobilisers / ignition keys
- Gate and depot access (especially rural and industrial)
- Tool cribs and workshop drawers
- Some older alarm systems

## Flipper Capabilities
- Read / Write / Emulate most DS1990A / DS1990R / RW1990 type iButtons
- Clone many simple ones in seconds
- Some "encrypted" or challenge-response variants exist (harder)

## Recommended Workflow
1. Read the iButton with Flipper (iButton app → Read)
2. Save it
3. Test emulation on the actual reader before committing crimes against your own gear

## Notes from the Field (Australia)
Many cheap Chinese "universal" iButton writers are sold on eBay AU. They often just write the ID. The Flipper can do the same and more.

**Legal reminder:** Only on hardware you own or have explicit written permission to test.
